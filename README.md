## What is BisQue

[BisQue](https://bisque2.ece.ucsb.edu/client\_service/) is an open-source web-based platform that supports **data storage for petabytes, ML based data analysis and reproducible workflows**, and **visualization**.



<figure><img src=".gitbook/assets/image (5) (1).png" alt=""><figcaption></figcaption></figure>

BisQue is specifically designed to provide researchers with organizational and quantitative analysis tools for up to 5D image data. Users can extend BisQue by creating their own modules that take advantage of cutting edge machine learning algorithms. BisQueâ€™s extensibility stems from two core concepts: an open web-based architecture and a flexible metadata facility. Together, these empower researchers to create, develop, and share novel multi-modal data analyses.
